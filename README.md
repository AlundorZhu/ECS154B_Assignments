## Starting locally
Clone the repo with `git clone --recurse-submodules <repo_url>`. 

Download [docker](https://www.docker.com/get-started/), install devcontainer extension in vscode, open the clone repo in vscode, and click "Reopen in Container" when prompted. You should be ready to go after the container is built.
